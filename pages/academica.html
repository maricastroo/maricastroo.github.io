Meus Projetos

LYRIC DEFENDER

Jogo desenvolvido na plataforma Construct 3 da empresa Scirra Ltd. Projeto feito para a matéria de Experiência Criativa no 1º período!

O objetivo do jogo é que o jogador consiga desviar dos obstáculos enquanto digita a letra da música para acertar o inimigo. O jogo contém 5 fases, ao chegar na última o jogador enfrenta o terrível e temido Boss!

Fenda do Biquíni Canecas

Fenda do biquíni canecas é uma loja de canecas online (fictícia). Desenvolvida para a matéria Programação WEB no 2º período!

Este é um projeto acadêmico fictício que simula uma loja virtual de canecas personalizadas inspiradas no universo do Bob Esponja. O objetivo do projeto foi integrar diferentes tecnologias web — HTML, CSS, JavaScript e PHP — com um banco de dados MySQL para criar uma aplicação web funcional.

Projeto POO

Projeto POO é um sistema fictício desenvolvido para a disciplina de Programação Orientada a Objetos no 3º período.

Este projeto acadêmico tem como objetivo aplicar os principais conceitos da programação orientada a objetos, como herança, encapsulamento e polimorfismo, por meio da construção de um sistema em Java que salva no Banco de Dados. A proposta foca em consolidar os fundamentos da POO com uma estrutura de classes bem definida e exemplos práticos de uso.

Urban Tech

Urban Tech é um sistema web desenvolvido como parte de um projeto acadêmico para a disciplina de Experiência Criativa no 3º período.

O projeto tem como objetivo oferecer uma solução digital para problemas urbanos, utilizando Python e o framework Flask. Com uma estrutura organizada em camadas (MVC), o sistema permite uma navegação simples e intuitiva, além de demonstrar práticas modernas de desenvolvimento web. A aplicação contempla funcionalidades básicas para gerenciar dados e exibir informações de forma interativa por meio de uma interface amigável. Utilizamos também banco de dados para armazenar as informações e o wokwi para simular o funcionamento do projeto com microprocessadores.
